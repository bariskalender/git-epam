Task 11. Correlated subqueries.

Task 3:
SELECT
    p.id,
    p.comment AS title,
    p.price
FROM product AS p
JOIN product_title AS pt
    ON pt.id = p.product_title_id
WHERE p.price > (
    SELECT AVG(p2.price)
    FROM product AS p2
    JOIN product_title AS pt2
        ON pt2.id = p2.product_title_id
    WHERE pt2.product_category_id = pt.product_category_id
)
ORDER BY p.id ASC;
