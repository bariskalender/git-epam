# EPAM Trainee Program Tasks and Applications

Tasks and applications I built while learning in the EPAM trainee program.

This repository collects my SQL, C#, .NET, ASP.NET Core, Entity Framework, testing, asynchronous programming, serialization, and Web API practice projects. Most folders include their own README files with task-specific notes and requirements.

## Repository Structure

- `sql/standalone-scripts` - standalone SQL scripts for database creation, CRUD operations, joins, views, grouping, and constraints.
- `sql/epam-sql-tasks` - SQL task folders from EPAM practice assignments.
- `dotnet` - C#, .NET, ASP.NET Core, Entity Framework, ADO.NET, testing, and Web API projects.

## Notes

Build outputs, local IDE files, package caches, binaries, and temporary files are intentionally excluded from the repository. The goal is to keep only the source code, SQL scripts, project files, and documentation needed to review the EPAM trainee program work.
